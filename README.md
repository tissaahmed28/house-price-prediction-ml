# House Price Prediction using Machine Learning

An end-to-end Machine Learning regression project that predicts house prices based on socio-economic and geographical features.  
The project includes data analysis, model training, evaluation, and an interactive Streamlit web application.

---

##  Demo

Interactive web app built with Streamlit 
Users can enter house features and instantly get a price prediction.

---

## Dataset

- California Housing Dataset
- Features include:
  - Median Income (×10,000$)
  - House Age
  - Average Rooms
  - Average Bedrooms
  - Population
  - Average Occupancy
  - Latitude & Longitude

---

## Machine Learning Models

- Linear Regression (baseline)
- Random Forest Regressor (final model)

### Evaluation Metrics:
- Mean Squared Error (MSE)
- Mean Absolute Error (MAE)
- R² Score
